What I learned

So, at the start I wanted to do it using python because it was pretty easier, but, how would I share with you if I did it with python? So, I decided to just go with pure js, html and css.

I used the canvas html element which give me a huuuuge help, it was pretty easy to quickly draw every planet while keeping track of each ones of theirs location using objects and lists. With that done, I just needed to calculate where the body is going to move, I didn't went really deep into this, just used the classical newtonian physics (F = G*m1*m2/r**2), doing some adjustments on this formula and combining it with (F = m * a) you can cancel out one of the masses and get that the acceleration of a body = G * m2 (mass of the other body) / r ** 2.

From there, multiply the acceleration that you got by the dt you are using, which gives how much a body velocity will change in the next frame, add that to the velocity, add the velocity * dt to the position, annnnd finaly you are done and can draw it on the canvas :D
