Lancerforge is a website made to make freelancers lives easier, by managing all their projects in just one place, it offers the functions of, project management and classification, task management, kanban, time and expenses tracking on each project, automatic calculation of costs and CRUD operations (Create, Read, Update and Delete). Its located at lancerforge.com

February - 2025

Some photos of it

What I learned

This was my first big project with programming, for it, I used Django and learnt everything from the basics, I could learn how to create models for the database, creating users, handling logins, and signing them in, merging python with html, css and javascript, deploying a real project into the web and also integrating the stripe API for the premium features of it. At all, I really enjoyed this project and was very happy to see people really using it on their day to day life, I got some reviews on discord and most of them liked it and said that it was useful for them.


Techs I used

  • Python with Django
  • HTML
  • CSS
  • Javascript
  • Tailwind CSS
  • PostgreSQL
  • Gunicorn
  • Nginx
  • Stripe API
  • Hostinger for deploying

Challenges and Solutions

I think that my biggest challenge on this project was linking the stripe API and making it work, because it works by sending a lot of different types of requests, you need to read them, identify the type and see what you need to do with it, maybe its updating an user to premium, removing the premium for a user, deleting the user... Also, everytime you create an user on your database you need to create a customer for it on the stripe, what made it even more difficult.
